## Parte teórica (10 puntos)
1. **(2 puntos)** Define una función como una relación entre dos conjuntos.  
2. **(2 puntos)** ¿Qué es el dominio de una función? ¿Y el rango?  
3. **(2 puntos)** Explica con tus palabras qué significa que una función sea **inyectiva**.  
4. **(2 puntos)** ¿Cuándo una función es **sobreyectiva**?  
5. **(2 puntos)** Escribe la forma general de una función afín e identifica el significado de cada parámetro.

## Parte práctica (10 puntos)
Dada la función (función) encontrar el valor de la misma para el valor de x indicado.

| Función | Valor a Encontrar | Tarea: Encontrar... |
|---|---|---|
| 1. f(x) = 5x - 3 | f(2) | El valor de f cuando x=2. |
| 2. g(x) = -2x + 7 | g(-4) | El valor de g cuando x=-4. |
| 3. h(x) = x² + 1 | h(3) | El valor de h cuando x=3. |
| 4. j(x) = x² - 5x | j(-1) | El valor de j cuando x=-1. |
| 5. k(x) = 3x² - 2x + 1 | k(0) | El valor de k cuando x=0. |
